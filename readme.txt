== Installation ==

1. Upload the `advance-slider` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the `[AWP-SLIDER]` shortcode to add sliders to your posts or pages.
